    <h1>Overview of Machine Learning</h1>
    <p>
        <span class="accent-2">Machine Learning (ML)</span> is a branch of <span class="accent">artificial intelligence</span> where computers <span class="accent-3">learn patterns from data</span> and then use those patterns to <strong class="accent">make decisions, predictions, or classifications</strong> — <em>without being explicitly programmed</em> for every scenario.
    </p>

    <h3>Data</h3>
    <p>
       Data is simply the <span class="accent">information a computer learns from</span>. It can be <strong class="muted-strong">numbers, text, images, sounds</strong>—anything that helps the model <span class="accent-3">recognize patterns</span> and <strong class="accent">make predictions</strong>.
    </p>

    <h3>Models</h3>
    <p>
        A model is a <strong class="muted-strong">mathematical or computational structure</strong> that <strong>makes predictions or decisions</strong> based on what it has <strong class="accent-3">learned from data</strong>.
    </p>

    <div class="callout">
        <span class="label">Example:</span> A recipe the computer creates after studying lots of examples
    </div>

    <h3>Types of Models</h3>
    <p>
        ML systems fall into one or more of the following categories based on how they <strong class="accent">learn to make predictions</strong>:
    </p>

    <ul>
        <li>Supervised learning</li>
        <li>Unsupervised learning</li>
        <li>Reinforcement learning</li>
        <li>Generative AI</li>
    </ul>

    <h1>Supervised Learning</h1>
    <p>
        <strong class="accent-2">Supervised learning</strong> is a type of machine learning where the model <strong class="accent">learns from labeled data</strong> — meaning each <strong class="accent-3">input comes with the correct output</strong> already provided.
    </p>
