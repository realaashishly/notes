    <h1>Overview of Machine Learning</h1>
    <p>
        <span class="accent-2">Machine Learning (ML)</span> is a branch of <span class="accent">artificial intelligence</span> where computers <span class="accent-3">learn patterns from data</span> and then use those patterns to <strong class="accent">make decisions, predictions, or classifications</strong> — <em>without being explicitly programmed</em> for every scenario.
    </p>

    <h3>Data</h3>
    <p>
       Data is simply the <span class="accent">information a computer learns from</span>. It can be <strong class="muted-strong">numbers, text, images, sounds</strong>—anything that helps the model <span class="accent-3">recognize patterns</span> and <strong class="accent">make predictions</strong>.
    </p>

    <h3>Models</h3>
    <p>
        A model is a <strong class="muted-strong">mathematical or computational structure</strong> that <strong>makes predictions or decisions</strong> based on what it has <strong class="accent-3">learned from data</strong>.
    </p>

    <div class="callout">
        <span class="label">Example:</span> A recipe the computer creates after studying lots of examples
    </div>

    <h3>Types of Models</h3>
    <p>
        ML systems fall into one or more of the following categories based on how they <strong class="accent">learn to make predictions</strong>:
    </p>

    <ul>
        <li>Supervised learning</li>
        <li>Unsupervised learning</li>
        <li>Reinforcement learning</li>
        <li>Generative AI</li>
    </ul>

    <h1>Supervised Learning</h1>
    <p>
        <strong class="accent-2">Supervised learning</strong> is a type of machine learning where the model <strong class="accent">learns from labeled data</strong> — meaning each <strong class="accent-3">input comes with the correct output</strong> already provided.
    </p>

    <h2>Most Common Use Cases for Supervised Learning</h2>
    <p>
      Supervised learning problems are generally grouped into two categories:
      <span class="text-red">Regression</span> and
      <span class="text-blue">Classification</span>.
    </p>



    <img src="/ml/1.png" alt="Regression vs Classification" class="diagram">


    <h3>1. Regression</h3>
    <p>
      A regression model predicts a
      <span class="bg-red">numeric value</span> (continuous data).
    </p>
    <ul>
      <li>
        Used when the output variable is a real number, like
        <span class="text-red">price, weight, or height</span>.
      </li>
    </ul>

    <div class="example-box">
      <span class="example-label">Example</span> A weather model that predicts
      the <strong>amount of rain</strong> (in inches or millimeters).
    </div>

    <hr />

    <h3>2. Classification</h3>
    <p>
      Classification models predict the likelihood that something
      <span class="bg-blue">belongs to a category</span> (discrete classes).
    </p>
    <ul>
      <li>
        Used when the output variable is a category, such as
        <span class="text-blue">"Yes/No"</span> or
        <span class="text-blue">"Spam/Not Spam"</span>.
      </li>
    </ul>

    <div class="example-box">
      <span class="example-label">Examples</span> Predicting if an email is
      <strong>spam</strong> or identifying if a photo contains a
      <strong>cat</strong>.
    </div>

    <h2>Unsupervised Learning</h2>
    <p>
      <span class="text-green">Unsupervised learning</span> is a type of machine
      learning where the model
      <span class="bg-green">learns without labeled data</span>.
    </p>



    <img src="/ml/2.png" alt="Unsupervised Learning" class="diagram">


    <p>
      This means that
      <span class="text-green">no output/label input is provided</span> to guide
      the learning process; the model must find patterns on its own.
    </p>
    <h3>Clustering</h3>
    <p>
      A commonly used <span class="text-green">unsupervised learning model</span>
      employs a technique called
      <span class="bg-purple">clustering</span>.
    </p>

    <div class="diagram-container">
    <img src="/ml/3.png" alt="Clustering" class="diagram">
    <img src="/ml/4.png" alt="Clustering" class="diagram">
    </div>

    <h2>Reinforcement Learning</h2>
    <p>
      <span class="text-orange">Reinforcement learning</span> models make
      predictions by getting
      <span class="bg-orange">rewards or penalties</span> based on actions
      performed within an environment.
    </p>

    <img src="/ml/5.png" alt="Reinforcement Learning" class="diagram">

    <h2>Generative AI</h2>
    <p>
      Generative AI is a subset of artificial intelligence capable of
      <span class="bg-teal">creating new content</span>, including text, images,
      audio, and video.
    </p>
    
    

    <p>
      Unlike traditional AI which classifies data, Generative AI learns patterns
      from training data to <span class="text-teal">generate novel outputs</span>
      that resemble the original data but are unique.
    </p>

    <div class="example-box">
      <span class="example-label">Examples</span> ChatGPT (Text), DALL-E &
      Midjourney (Images), Sora (Video).
    </div>
