<h1>NumPy UltraQuick Tutorial</h1>
    <p>
      <span class="bg-blue">NumPy</span> is a Python library for creating and
      manipulating <span class="text-blue">matrices</span>, the main data
      structure used by ML algorithms. Matrices are mathematical objects used to
      store values in rows and columns.
    </p>

    

    <h3>Terminology</h3>
    <ul>
      <li><strong>Python</strong> calls matrices <span class="text-red">lists</span>.</li>
      <li><strong>NumPy</strong> calls them <span class="text-green">arrays</span>.</li>
      <li><strong>TensorFlow</strong> calls them <span class="text-purple">tensors</span>.</li>
    </ul>

    <hr />

    <h2>1. Import NumPy Module</h2>
    <p>Run the following code cell to import the NumPy module:</p>
    <div class="code-block">import numpy as np</div>

    <hr />

    <h2>2. Populate Arrays</h2>
    <h3>Specific Numbers</h3>
    <p>
      Call <span class="bg-green">np.array</span> to create a NumPy array with
      your own hand-picked values.
    </p>

    <div class="code-block">
      <span class="comment"># One-dimensional array</span><br />
      one_dimensional_array = np.array([1.2, 2.4, 3.5, 4.7, 6.1, 7.2, 8.3, 9.5])<br />
      print(one_dimensional_array)
    </div>

    <p>
      To create a <span class="text-green">two-dimensional array</span>, specify
      an extra layer of square brackets.
    </p>

    <div class="code-block">
      <span class="comment"># Two-dimensional array (3x2)</span><br />
      two_dimensional_array = np.array([[6, 5], [11, 7], [4, 8]])<br />
      print(two_dimensional_array)
    </div>

    <div class="example-box">
      <span class="example-label">Tip</span> To populate an array with all
      zeros, call <strong>np.zeros</strong>. For all ones, call
      <strong>np.ones</strong>.
    </div>

    <hr />

    <h3>Sequences of Numbers</h3>
    <p>
      You can populate an array with a sequence using
      <span class="bg-purple">np.arange</span>.
    </p>

    <div class="code-block">
      sequence_of_integers = np.arange(5, 12)<br />
      print(sequence_of_integers)
    </div>

    <div class="example-box">
      <span class="example-label">Note</span> np.arange includes the
      <span class="text-purple">lower bound (5)</span> but
      <span class="text-red">not the upper bound (12)</span>.
    </div>

    <hr />

    <h3>Random Numbers</h3>
    <p>
      NumPy provides various functions to populate arrays with random numbers
      across certain ranges.
    </p>
    <ul>
      <li>
        <span class="bg-orange">np.random.randint</span>: Generates random
        integers between a low and high value.
      </li>
      <li>
        <span class="bg-teal">np.random.random</span>: Generates random
        floating-point values between 0.0 and 1.0.
      </li>
    </ul>

    <div class="code-block">
      <span class="comment"># Random integers between 50 and 100</span><br />
      random_integers_between_50_and_100 = np.random.randint(low=50, high=101,
      size=(6,))<br />
      print(random_integers_between_50_and_100)
    </div>

    <div class="code-block">
      <span class="comment"># Random floats between 0.0 and 1.0</span><br />
      random_floats_between_0_and_1 = np.random.random((6,))<br />
      print(random_floats_between_0_and_1)
    </div>

    <hr />

    <h2>3. Mathematical Operations & Broadcasting</h2>
    <p>
      If you want to add, subtract, or multiply two arrays, linear algebra
      usually requires operands to have compatible dimensions.
    </p>
    <p>
      NumPy uses a trick called <span class="bg-red">Broadcasting</span> to
      virtually expand the smaller operand to dimensions compatible for linear
      algebra.
    </p>

    

    <div class="code-block">
      <span class="comment"># Adds 2.0 to every item in the array</span><br />
      random_floats_between_2_and_3 = random_floats_between_0_and_1 + 2.0<br />
      print(random_floats_between_2_and_3)
    </div>

    <div class="code-block">
      <span class="comment"># Multiplies each cell in the array by 3</span><br />
      random_integers_between_150_and_300 = random_integers_between_50_and_100 *
      3<br />
      print(random_integers_between_150_and_300)
    </div>

    <h1>Tasks</h1>

    <h3>Task 1: Create a Linear Dataset</h3>
    <p>
      <strong>Goal:</strong> Create a simple dataset consisting of a single
      feature and a label.
    </p>
    <ul>
      <li>
        Assign a sequence of integers from 6 to 20 (inclusive) to an array named
        <span class="text-blue">feature</span>.
      </li>
      <li>
        Assign 15 values to an array named
        <span class="text-blue">label</span> such that:
        <span class="bg-purple">label = (3)(feature) + 4</span>
      </li>
    </ul>

    <div class="code-block">
      feature = np.arange(6, 21)<br />
      print(feature)<br />
      label = (feature * 3) + 4<br />
      print(label)
    </div>

    <hr />

    <h3>Task 2: Add Some Noise</h3>
    <p>
      To make the dataset realistic, modify each value assigned to label by
      adding a different <span class="text-orange">random floating-point value</span> 
      between -2 and +2.
    </p>
    <div class="example-box">
      <span class="example-label">Constraint</span> Don't rely on broadcasting.
      Create a noise array having the same dimension as label.
    </div>

    <div class="code-block">
      noise = (np.random.random((15,)) * 4) - 2<br />
      print(noise)<br />
      label = label + noise<br />
      print(label)
    </div>
